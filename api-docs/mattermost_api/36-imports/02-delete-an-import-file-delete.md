# 02-Delete an import file [DELETE]

`DELETE /api/v4/imports/{import_name}`

Deletes an import file.


__Minimum server version__: 5.31

##### Permissions

Must have `manage_system` permissions.


### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `import_name` | `string` | `path` | Yes | The name of the import file to delete |
