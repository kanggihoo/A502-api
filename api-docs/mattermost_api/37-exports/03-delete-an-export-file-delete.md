# 03-Delete an export file [DELETE]

`DELETE /api/v4/exports/{export_name}`

Deletes an export file.


__Minimum server version__: 5.33

##### Permissions

Must have `manage_system` permissions.


### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `export_name` | `string` | `path` | Yes | The name of the export file to delete |
