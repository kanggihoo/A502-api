# 02-Download an export file [GET]

`GET /api/v4/exports/{export_name}`

Downloads an export file.


__Minimum server version__: 5.33

##### Permissions

Must have `manage_system` permissions.


### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `export_name` | `string` | `path` | Yes | The name of the export file to download |
