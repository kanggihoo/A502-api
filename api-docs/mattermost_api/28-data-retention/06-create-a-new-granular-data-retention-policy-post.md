# 06-Create a new granular data retention policy [POST]

`POST /api/v4/data_retention/policies`

Creates a new granular data retention policy with the specified display
name and post duration.

__Minimum server version__: 5.35

##### Permissions
Must have the `sysconsole_write_compliance_data_retention` permission.

##### License
Requires an E20 license.


### Request Body (application/json)

```json
any
```
### Responses

#### 201 - Retention policy successfully created.
