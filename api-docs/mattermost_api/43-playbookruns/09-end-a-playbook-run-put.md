# 09-End a playbook run [PUT]

`PUT /plugins/playbooks/api/v0/runs/{id}/end`

End a playbook run

### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `id` | `string` | `path` | Yes | ID of the playbook run to end. |

#### 200 - Playbook run ended

#### 500 -
