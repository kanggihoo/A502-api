# 06-Get custom emoji image [GET]

`GET /api/v4/emoji/{emoji_id}/image`

Get the image for a custom emoji.
##### Permissions
Must be authenticated.


### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `emoji_id` | `string` | `path` | Yes | Emoji GUID |
