# 04-Download a report [GET]

`GET /api/v4/compliance/reports/{report_id}/download`

Download the full contents of a report as a file.
##### Permissions
Must have `manage_system` permission.


### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `report_id` | `string` | `path` | Yes | Compliance report GUID |

### Responses

#### 200 - The compliance report file

#### 400 - 

#### 401 - 

#### 403 - 

#### 501 -
