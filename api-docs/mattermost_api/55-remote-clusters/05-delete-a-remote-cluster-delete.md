# 05-Delete a remote cluster. [DELETE]

`DELETE /api/v4/remotecluster/{remote_id}`

Deletes a Remote Cluster.

##### Permissions
`manage_secure_connections`

| --- | --- | --- | --- | --- |
| `remote_id` | `string` | `path` | Yes | Remote Cluster GUID |
