# 05-Get thumbnail metadata headers [HEAD]

`HEAD /api/v4/files/{file_id}/thumbnail`

Performs the same permission and existence checks as getting a thumbnail, but returns headers only.


### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `file_id` | `string` | `path` | Yes | The ID of the file to get |
