# 11-Get public file metadata headers [HEAD]

`HEAD /files/{file_id}/public`

Performs the same validation checks as getting a public file, but returns headers only.


### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `file_id` | `string` | `path` | Yes | The ID of the file to get |
| `h` | `string` | `query` | Yes | File hash |
