# 09-Delete plan-only team [DELETE]

`DELETE /rest/api/3/plans/plan/{planId}/team/planonly/{planOnlyTeamId}`

Deletes a plan-only team and their planning settings.

**[Permissions](#permissions) required:** *Administer Jira* [global permission](https://confluence.atlassian.com/x/x4dKLg).

### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `planId` | `integer` | `path` | Yes | The ID of the plan. |
| `planOnlyTeamId` | `integer` | `path` | Yes | The ID of the plan-only team. |
