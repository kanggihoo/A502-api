# Project types

This resource represents project types. Use it to obtain a list of all project types, a list of project types accessible to the calling user, and details of a project type.

- [01-Get all project types [GET]](./01-get-all-project-types-get.md)

- [03-Get project type by key [GET]](./03-get-project-type-by-key-get.md)
- [04-Get accessible project type by key [GET]](./04-get-accessible-project-type-by-key-get.md)
