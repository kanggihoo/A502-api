# 10-Delete a license [DEL]

`DELETE /api/v4/license/{id}`

Deletes a specified license.

### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `id` | `integer` | `path` | Yes | ID of the GitLab license |

### Responses

#### 204 - Resource deleted

#### 400 - Bad Request

#### 404 - Not found
