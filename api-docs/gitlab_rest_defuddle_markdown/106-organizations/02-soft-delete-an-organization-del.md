# 02-Soft-delete an organization [DEL]

`DELETE /api/v4/organizations/{id}`

This feature was introduced in GitLab 19.2.

### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `id` | `integer` | `path` | Yes | The ID of the organization |

### Responses

#### 202 - Accepted

#### 400 - Bad request

#### 401 - Unauthorized

#### 403 - Forbidden

#### 404 - Not found
