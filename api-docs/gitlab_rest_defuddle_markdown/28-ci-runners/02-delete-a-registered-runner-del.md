# 02-Delete a registered runner [DEL]

`DELETE /api/v4/runners`

Deletes a specified registered runner.

### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `token` | `string` | `query` | Yes | The runner's authentication token |

### Responses

#### 204 - Resource deleted

#### 400 - Bad Request

#### 403 - Forbidden
