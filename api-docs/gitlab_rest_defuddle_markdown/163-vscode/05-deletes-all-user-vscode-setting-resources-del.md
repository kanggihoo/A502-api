# 05-Deletes all user vscode setting resources [DEL]

`DELETE /api/v4/vscode/settings_sync(/{settings_context_hash})/v1/collection`

Deletes all user vscode setting resources

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `settings_context_hash` | `any` | `path` | Yes |  |
