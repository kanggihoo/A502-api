# Members

  ### 1. Members API

  • 유용성: P0 (핵심). 팀원 목록 조회(03, 10번), 팀 권한 판별(14번), 팀 생성 시 팀원 추가(09번).
  • 제약 사항: Billable members(21~23번)는 유료 티어 전용으로 제외.
