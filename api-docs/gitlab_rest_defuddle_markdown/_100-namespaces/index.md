# Namespaces
  ### 5. Namespaces API

  • 유용성: P1 (조회 유용).
      • List / Retrieve namespaces (03, 07번): 유저/그룹 네임스페이스 식별 및 URL 경로 파악.
      • Storage limit exclusions (04~06번): 인스턴스 최고 관리자(Admin) 전용.
  • 티어/권한: Free / CE 지원. Developer 이상 가능.
