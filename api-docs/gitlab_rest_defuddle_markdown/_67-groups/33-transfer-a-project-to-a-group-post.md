# 33-Transfer a project to a group [POST]

`POST /api/v4/groups/{id}/projects/{project_id}`

Transfers a specified project to another group. Administrators only.

### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `id` | `string` | `path` | Yes | The ID of a group |
| `project_id` | `string` | `path` | Yes | The ID or path of the project |
