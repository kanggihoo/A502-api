# 08-Get a raw snippet [GET]

`GET /api/v4/snippets/{id}/raw`

This feature was introduced in GitLab 8.15.

### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `id` | `integer` | `path` | Yes | The ID of a snippet |

### Responses

#### 200 - OK

#### 400 - Bad Request

#### 404 - Not found
