# 06-Delete an instance member role [DEL]

`DELETE /api/v4/member_roles/{member_role_id}`

Deletes an instance member role.

### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `member_role_id` | `integer` | `path` | Yes | ID of the member role |

### Responses

#### 204 - 204 No Content

#### 400 - Bad Request

#### 401 - Unauthorized

#### 404 - 404 Member Role Not Found
