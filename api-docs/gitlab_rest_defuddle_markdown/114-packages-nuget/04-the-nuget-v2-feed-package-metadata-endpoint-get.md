# 04-The NuGet V2 Feed Package $metadata endpoint [GET]

`GET /api/v4/projects/{id}/packages/nuget/v2/$metadata`

This feature was introduced in GitLab 16.3

### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `id` | `any` | `path` | Yes | The ID or URL-encoded path of the project |

### Responses

#### 200 - OK

#### 400 - Bad Request

#### 404 - Not Found
