# 14-The NuGet V2 Feed Service Index [GET]

`GET /api/v4/groups/{id}/-/packages/nuget/v2`

This feature was introduced in GitLab 16.2

### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `id` | `any` | `path` | Yes | The group ID or full group path. |

#### 200 - OK
