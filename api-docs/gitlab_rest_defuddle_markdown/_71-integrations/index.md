# Integrations

Operations related to integrations.
  • 유용성: P1 (조건부 유용 - 내장 연동 설정).
      • Jenkins integration (25, 80, 135번): GitLab 프로젝트 내 젠킨스 서비스 연동 설정.
      • Jira integration (26, 81, 136번): GitLab - Jira 내장 연동 설정.
      • Mattermost integration (43, 98, 153번): GitLab 내장 Mattermost 알림 연동.
  • 티어/권한: Free / CE 지원. 프로젝트 Maintainer 이상 권한 필요.
  • 특이사항:
      • GitLab 내장 연동을 API로 자동 설정할 수 있음.
      • 단, 커스텀 Webhook(Hooks API) + 통합 서버를 구축하는 편이 알림 메시지 커스텀, 링크 가공,
      커스텀 로직 적용에 훨씬 유리함.
  • 결론: 팀장(Maintainer) 토큰으로 내장 연동 자동화 시 활용 가능 (P1). 커스텀 알림 구축 시에는
  Hooks API 우선 사용 추천.
