# 05-Download dependency list export [GET]

`GET /api/v4/dependency_list_exports/{export_id}/download`

Downloads a dependency list export.

### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `export_id` | `any` | `path` | Yes | The ID of the dependency list export |

### Responses

#### 200 - OK

#### 404 - Not Found
