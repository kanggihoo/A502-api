# 06-Get vulnerabilities [GET]

`GET /api/v4/occurrences/vulnerabilities`

Returns vulnerabilities related to an occurrence.

### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `id` | `string` | `query` | Yes | The ID of the occurrence |

### Responses

#### 200 - OK

#### 400 - Bad Request
