# 03-Download a group avatar [GET]

`GET /api/v4/groups/{id}/avatar`

Downloads a group avatar image.

### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `id` | `string` | `path` | Yes | The ID of the group |

### Responses

#### 200 - OK

#### 400 - Bad Request

#### 404 - Not Found
