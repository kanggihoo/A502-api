# 02-Add code review comments to a merge request [POST]

`POST /api/v4/ai/duo_workflows/code_review/add_comments`

Adds review comments generated by Duo Workflows to a merge request.

### Request Body (application/json)

```json
{
  "project_id": string (required), // The ID or path of the project
  "merge_request_iid": integer (required), // The IID of the merge request
  "review_output": string (required), // The review output from LLM
  "workflow_id": integer, // The ID of the Duo Workflow session
}
```
### Responses

#### 200 - OK

#### 400 - Validation failed

#### 401 - Unauthorized

#### 403 - Forbidden

#### 404 - Not found
