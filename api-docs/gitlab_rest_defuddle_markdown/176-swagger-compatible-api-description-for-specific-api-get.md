# 176-Swagger compatible API description for specific API [GET]

`GET /api/v4/swagger_doc/{name}`

Swagger compatible API description for specific API

### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `name` | `string` | `path` | Yes | Resource name of mounted API |
| `locale` | `string` | `query` | No | Locale of API documentation |

### Responses

#### 200 - OK

#### 400 - Bad Request

#### 404 - Not Found
