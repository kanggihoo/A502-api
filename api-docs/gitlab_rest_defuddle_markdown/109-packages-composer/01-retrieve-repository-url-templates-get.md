# 01-Retrieve repository URL templates [GET]

`GET /api/v4/group/{id}/-/packages/composer/packages`

Retrieves the repository URL templates for requesting individual packages for a group.

### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `id` | `any` | `path` | Yes | The ID or URL-encoded path of a group |

### Responses

#### 200 - OK

#### 400 - Bad Request

#### 401 - Unauthorized

#### 404 - Not Found
