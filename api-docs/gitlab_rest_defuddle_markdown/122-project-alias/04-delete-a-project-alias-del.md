# 04-Delete a project alias [DEL]

`DELETE /api/v4/project_aliases/{name}`

Deletes a specified project alias. Administrators only.

### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `name` | `any` | `path` | Yes |  |
