# 02-List all packages for a project [GET]

`GET /api/v4/projects/{id}/packages/pypi/simple`

Lists all packages for a specified project in an HTML file.

### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `id` | `any` | `path` | Yes | The ID or URL-encoded path of the project |

### Responses

#### 200 - OK

#### 400 - Bad Request

#### 401 - Unauthorized

#### 403 - Forbidden

#### 404 - Not Found
