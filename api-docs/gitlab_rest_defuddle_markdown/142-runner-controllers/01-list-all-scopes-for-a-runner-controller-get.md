# 01-List all scopes for a runner controller [GET]

`GET /api/v4/runner_controllers/{id}/scopes`

Lists all scopes for a runner controller.

### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `id` | `integer` | `path` | Yes | ID of the runner controller |

### Responses

#### 200 - OK

#### 400 - Bad Request

#### 403 - Forbidden

#### 404 - Not found
