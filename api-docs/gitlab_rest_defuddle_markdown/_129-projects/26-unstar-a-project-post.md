# 26-Unstar a project [POST]

`POST /api/v4/projects/{id}/unstar`

Unstars a specified project.

### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `id` | `any` | `path` | Yes | The ID or URL-encoded path of the project |
