# Projects

  ### 1. Projects API

  • 유용성: P0 (최우선 핵심 / 전체 워크스페이스의 근간).
      • List/Retrieve projects (12, 16번): 대시보드 기준 엔터티, 팀 프로젝트 홈 링크 및 정보 조회.
      • Create project (13번): 팀 생성 자동화 흐름에서 전용 저장소 자동 생성 (팀장 토큰).
      • Project issues (64~67번): 프로젝트 단위 이슈 목록, 통계 및 생성/수정.
      • Programming language usage (28번): 프로젝트 주 사용 언어 타임라인/기술 스택 파악.
  • 제약 사항:
      • Audit events (41, 42번): GitLab Premium/Ultimate (유료 티어) 전용.
      • Runner 관련 (74~78번): Admin/Owner 권한 필요 및 젠킨스 사용으로 제외.
  • 티어/권한: 기본 기능 Free / CE 지원. Developer (조회) / Maintainer (생성/수정).
