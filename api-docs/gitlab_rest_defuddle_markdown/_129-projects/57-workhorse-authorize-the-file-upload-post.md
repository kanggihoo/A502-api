# 57-Workhorse authorize the file upload [POST]

`POST /api/v4/projects/{id}/uploads/authorize`

This feature was introduced in GitLab 13.11

### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `id` | `any` | `path` | Yes | The ID or URL-encoded path of the project |

### Responses

#### 200 - OK

#### 400 - Bad Request

#### 404 - Not found
