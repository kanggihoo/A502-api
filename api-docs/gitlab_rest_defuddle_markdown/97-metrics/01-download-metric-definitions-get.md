# 01-Download metric definitions [GET]

`GET /api/v4/usage_data/metric_definitions`

Downloads all metric definitions as a single YAML file.

### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `include_paths` | `boolean` | `query` | No | Include file paths in the metric definitions |

### Responses

#### 200 - OK

#### 400 - Bad Request

#### 401 - Unauthorized

#### 404 - Not found
