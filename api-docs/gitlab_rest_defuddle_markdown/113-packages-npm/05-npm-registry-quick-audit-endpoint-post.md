# 05-NPM registry quick audit endpoint [POST]

`POST /api/v4/packages/npm/-/npm/v1/security/audits/quick`

This feature was introduced in GitLab 15.6

### Responses

#### 200 - Ok

#### 307 - Temporary Redirect

#### 400 - Bad Request

#### 401 - Unauthorized

#### 403 - Forbidden

#### 404 - Not Found
