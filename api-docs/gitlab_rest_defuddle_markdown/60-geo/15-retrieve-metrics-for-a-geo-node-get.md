# 15-Retrieve metrics for a Geo node [GET]

`GET /api/v4/geo_nodes/{id}/status`

Retrieves metrics for a specified Geo node.

### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `id` | `integer` | `path` | Yes |  |
| `refresh` | `boolean` | `query` | No | Attempt to fetch the latest status from the Geo node directly, ignoring the cache |
