# 14-Delete a Geo node [DEL]

`DELETE /api/v4/geo_nodes/{id}`

Deletes a specified Geo node.

### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `id` | `integer` | `path` | Yes |  |

### Responses

#### 204 - 204 No Content

#### 400 - 400 Bad request

#### 401 - 401 Unauthorized

#### 403 - 403 Forbidden
