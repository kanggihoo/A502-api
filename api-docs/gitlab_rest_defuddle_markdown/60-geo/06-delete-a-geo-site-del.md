# 06-Delete a Geo site [DEL]

`DELETE /api/v4/geo_sites/{id}`

Deletes a specified Geo site.

### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `id` | `integer` | `path` | Yes |  |

### Responses

#### 204 - 204 No Content

#### 400 - 400 Bad request

#### 401 - 401 Unauthorized

#### 403 - 403 Forbidden

#### 404 - 404 GeoSite Not Found
