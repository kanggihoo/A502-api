# 03-Get offline transfer export details [GET]

`GET /api/v4/offline_exports/{id}`

Retrieves details of an offline transfer export

### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `id` | `integer` | `path` | Yes | The ID of user's offline transfer export |

### Responses

#### 200 - OK

#### 400 - Bad Request

#### 404 - Not Found
