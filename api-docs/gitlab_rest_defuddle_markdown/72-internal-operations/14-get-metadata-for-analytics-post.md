# 14-Get metadata for analytics. [POST]

`POST /api/v4/projects/{project_id}/product_analytics/request/meta`

Retrieves metadata for analytics from the cube installation

### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `project_id` | `integer` | `path` | Yes | ID of the project to get meta data |

### Responses

#### 201 - Created

#### 400 - Bad Request

#### 404 - Not Found
