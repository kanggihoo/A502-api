# 18-Delete a project [DEL]

`DELETE /api/v4/projects/{id}`

Deletes a specified project.

### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `id` | `any` | `path` | Yes | The ID or URL-encoded path of the project |

### Responses

#### 202 - Accepted

#### 400 - Bad Request

#### 403 - Unauthenticated

#### 404 - Not found
