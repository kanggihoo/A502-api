# 116-Create/Edit Asana integration [PUT]

`PUT /api/v4/groups/{id}/integrations/asana`

Set Asana integration.

### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `id` | `any` | `path` | Yes | The ID or URL-encoded path of the group |

### Request Body (application/json)

```json
{
  "api_key": string (required), // User API token. The user must have access to the task. All comments are attributed to this user.
  "restrict_to_branch": string, // Comma-separated list of branches to be automatically inspected. Leave blank to include all branches.
  "push_events": boolean, // Trigger event for pushes to the repository.
  "use_inherited_settings": boolean, // Indicates whether to inherit the default settings. Defaults to `false`.
}
```
