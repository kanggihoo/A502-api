# 05-Delete a project topic [DEL]

`DELETE /api/v4/topics/{id}`

Deletes a specified project topic. Administrators only.

### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `id` | `integer` | `path` | Yes | ID of project topic |

### Responses

#### 204 - Resource deleted

#### 400 - Bad Request

#### 404 - Not Found
