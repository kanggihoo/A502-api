# 55-Upload a recipe file [PUT]

`PUT /api/v4/packages/conan/v1/files/{package_name}/{package_version}/{package_username}/{package_channel}/{recipe_revision}/export/{file_name}`

Uploads a specified recipe file to the package registry. You must use the upload URL returned from the recipe upload URLs endpoint.

### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `package_name` | `string` | `path` | Yes | Package name |
| `package_version` | `string` | `path` | Yes | Package version |
| `package_username` | `string` | `path` | Yes | Package username |
| `package_channel` | `string` | `path` | Yes | Package channel |
| `recipe_revision` | `string` | `path` | Yes | Conan Recipe Revision |
| `file_name` | `string` | `path` | Yes | Package file name |

### Request Body (multipart/form-data)

```json
{
  "file": string (required), // The package file to be published (generated by Multipart middleware)
}
```
### Responses

#### 200 - OK

#### 400 - Bad Request

#### 401 - Unauthorized

#### 403 - Forbidden

#### 404 - Not Found
