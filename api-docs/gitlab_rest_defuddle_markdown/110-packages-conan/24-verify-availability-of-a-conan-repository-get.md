# 24-Verify availability of a Conan repository [GET]

`GET /api/v4/projects/{id}/packages/conan/v1/ping`

Verifies availability of a Conan repository.

### Parameters

| Name | Type | In | Required | Description |
| --- | --- | --- | --- | --- |
| `id` | `any` | `path` | Yes | The ID or URL-encoded path of the project |

### Responses

#### 200 - OK

#### 400 - Bad Request

#### 404 - Not Found
